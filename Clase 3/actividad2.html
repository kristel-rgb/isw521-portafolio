Programación en Ambiente Web I

En este curso se aprenden las bases para crear páginas web utilizando HTML, CSS y buenas prácticas de estructura semántica.

El objetivo principal es comprender cómo organizar correctamente el contenido de una página para que sea clara, accesible y fácil de mantener.

Relacionado

HTML Semántico

Uso correcto de etiquetas como header, nav, main, article, aside y footer.

CSS Básico

Aplicación de colores, tamaños, márgenes, bordes y distribución visual.
